# Todo App

## Description
A simple todo application built with Node.js, Express, and MongoDB.

## Features
- User authentication (signup/login)
- Create tasks
- Mark tasks as completed or deleted
- Filter tasks (pending/completed)

## Tech Stack
- Node.js
- Express
- MongoDB
- Mongoose

## Setup Instructions
1. Clone the repo
2. Run `npm install`
3. Add `.env` file
4. Run `npm start`
